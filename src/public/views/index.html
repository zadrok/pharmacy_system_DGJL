This is a project made by Lachlan Haggart, Dimitrija Lakovski, Gavin Rossiter and Jordan Boyle for a group assignment in Development Project 2: Design, Planning and Quality SWE30010

This is a system that is intended to allow a small business to track current stock and stock sales, with this data the system is able to generate a sales report.
